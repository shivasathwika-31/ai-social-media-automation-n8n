# AI Social Media Automation using n8n

## 📌 Project Overview

This project is an AI-powered social media automation workflow built using n8n.

The workflow automatically:

1. Detects article information from Google Sheets
2. Uses Google Gemini AI to analyze and summarize the article
3. Generates a LinkedIn post
4. Automatically publishes the post on LinkedIn

---

## 🔄 Workflow Architecture

Google Sheets Trigger
        ↓
Google Gemini AI
        ↓
Article Summary
        ↓
LinkedIn Post Generation
        ↓
LinkedIn Post Publishing

## ⚙️ Workflow Process

### 1. Google Sheets Trigger

The workflow receives article information from Google Sheets.

### 2. AI Article Analysis

Google Gemini AI analyzes the article and generates a concise summary containing important insights and actionable information.

### 3. LinkedIn Post Generation

The AI-generated summary is converted into a professional LinkedIn post.

### 4. LinkedIn Publishing

The final post is automatically published on LinkedIn.
